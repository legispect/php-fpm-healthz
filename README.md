# php-fpm-healthz

This is a simple server that wrapped with [renatomefi/php-fpm-healthcheck](https://github.com/renatomefi/php-fpm-healthcheck).
It is useful because external health probes in Kubernetes world usually performs HTTP request.

Please see the Dockerfile for further information.
